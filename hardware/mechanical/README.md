# Mechanical

This directory is for the practical installation details around the passive tap.

## Concerns to cover

- enclosure choice for the interface and ESP32
- cable strain relief
- safe routing of low-voltage wiring
- serviceability and easy removal

## Installation principles

- keep the tap accessible so it can be removed quickly
- avoid permanent-looking wiring until the MVP is proven
- keep prototype parts away from hot, damp, or high-vibration areas
- do not place prototype electronics inside mains-only compartments
