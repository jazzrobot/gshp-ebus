# Proto v1 Build Notes

Session notes, breadboard layout observations, and bring-up results go here.

## Breadboard layout hints

- Keep bus-side components (BR1, DZ2, R1/R2, LM393, R3–R7) on one half of the board
- Keep logic-side components (PC817 transistor, R8, R9) on the other half
- Do not run jumper wires between the two halves except through the PC817
- Label or colour-code the GND_EBUS and GND_LOGIC rails so they are never accidentally bridged
